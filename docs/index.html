ÔøΩÔøΩ Quick Start Guide

Classic Mode

from langchain_google_genai import ChatGoogleGenerativeAI
from darbot_windows_agent.agent import Agent
from dotenv import load_dotenv

load_dotenv()

# Initialize LLM
llm = ChatGoogleGenerativeAI(model='gemini-2.0-flash')

# Create agent with optional vision support
agent = Agent(llm=llm, browser='chrome', use_vision=True)

# Execute automation task
query = input('Enter your query: ')
result = agent.invoke(query=query)
print(result.content)

Enhanced Mode with GitHub Copilot

# Interactive model selection with GitHub Copilot support
python main_enhanced.py

# Features:
# - Interactive model selection menu
# - GitHub Copilot (GPT-4, GPT-4 Turbo, GPT-3.5 Turbo)
# - GitHub CLI commands integration
# - Multi-provider support (OpenAI, Google, Groq, Ollama)

Environment Setup

# Create .env file
GOOGLE_API_KEY=your_google_api_key
OPENAI_API_KEY=your_openai_api_key  # optional
GROQ_API_KEY=your_groq_api_key      # optional

# For GitHub Copilot (no API key needed)
# Install GitHub CLI: winget install GitHub.cli
# Authenticate: gh auth login

Agent Class

Agent(llm, browser='chrome', use_vision=False, model_selector=None, max_steps=20)

Main agent class for Windows automation with LLM integration

Parameters:

  • llm (BaseChatModel) - LangChain compatible language model instance
  • browser (str) - Default browser for web automation ('chrome', 'edge', 'firefox')
  • use_vision (bool) - Enable vision capabilities with screenshot analysis
  • model_selector (ModelSelector) - Optional model selector instance for multi-provider support
  • max_steps (int) - Maximum number of steps for task completion (default: 20)
agent.invoke(query: str) ‚Üí AgentResult

Execute a natural language automation query

Parameters:

  • query (str) - Natural language task description

Returns:

  • AgentResult - Result object containing execution output and metadata

Common questions and answers about Darbot Windows Agent

Darbot supports multiple LLM providers through LangChain integration:

  • GitHub Copilot - GPT-4, GPT-4 Turbo, GPT-3.5 Turbo (via GitHub authentication)
  • Google Gemini - 2.0 Flash, 1.5 Pro (recommended for best performance)
  • OpenAI GPT - Direct API access to GPT-4, GPT-3.5 Turbo, and other models
  • Groq - Llama 3.1 70B, Mixtral 8x7B for fast inference
  • Ollama - Run local models like Llama 2, Mistral, Code Llama
  • Anthropic Claude - Claude 3 models via LangChain

All providers are interchangeable through the unified Agent interface.

Performance Metrics:

  • Average Total: 1.5-2.3 seconds per action
  • UI State Capture: ~100-200ms
  • LLM Processing: 800-1500ms (model-dependent)
  • Action Execution: ~200-500ms

Factors Affecting Performance:

  • LLM model choice (GPT-4 vs Gemini 2.0 Flash vs local models)
  • Network latency to LLM API
  • UI complexity (number of elements in window)
  • System resources (CPU, memory availability)

Darbot Windows Agent is built with AutoGen 2.0 (ag2ai) principles in mind:

  • Tool Composition: Composable tools following ag2ai patterns for building complex workflows
  • Multi-Agent Coordination: Support for coordinated multi-agent workflows and task delegation
  • Semantic Understanding: Context-aware action planning using semantic understanding of UI and tasks
  • Extensible Framework: Plugin architecture compatible with ag2ai tool standards
  • State Management: Persistent state tracking across agent invocations for complex tasks

The agent can be integrated into larger AutoGen-based systems for sophisticated automation pipelines.

Multi-Agent Considerations:

  • ‚úÖ Different Machines: Multiple agents can run on separate machines simultaneously
  • ‚ö†Ô∏è Same Machine: Agents may conflict for UI control on a single desktop
  • üí° Coordination: Implement coordination logic using AutoGen 2.0 patterns
  • üí° Task Queuing: Use async/await patterns for sequential task execution
  • üí° Virtual Desktops: Windows 10/11 virtual desktops can help isolate agents

For complex multi-agent scenarios, consider using AutoGen 2.0 (ag2ai) coordination patterns.
